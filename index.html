EyeRest

A modern desktop app that reminds you to take regular breaks for your eye health.

1. Installation

To start using EyeRest, follow these steps:

  1. Go to the GitHub Releases page:
    Releases - EyeRest v.1.0.0
  2. Download the eyerest-app-react.zip file and extract it to a folder on your computer.
  3. Double-click the EyeRest.exe file inside the folder to launch the application.

2. Main Screen

When you open the app, you will see the main screen. Here you can start the timer or skip the break.

3. Menu

Click the three dots icon at the top right to access settings, language options, and quit the app.

4. Time Settings

In the "Time settings" section, you can set the break interval. Enter the duration in minutes and save.

5. Theme Selection

In the "Theme settings" section, you can change the app's color theme. Choose one of the color options.

6. Break Screen

When it's time for a break, suggestions for resting your eyes will be shown. Click the "Rested, start timer again!" button to restart the timer.
